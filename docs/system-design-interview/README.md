# System Design Interview

Fast review material for system-design and architecture interviews.

Focus:

- recall tradeoffs quickly
- avoid blanking on concepts you already know
- answer like a senior engineer: context, choice, cost, and risk

## How to use it

- start with `00-answer-framework.md`
- review topics by subject
- use `08-common-questions.md` for quick drilling
