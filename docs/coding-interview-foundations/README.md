# Coding Interview Foundations

A simple, non-intimidating track for coding-challenge and coding-interview problems in general.

This material is meant for people who can build real software but still blank on algorithm interview patterns under pressure.

## Goal

- remove the "this is only for geniuses" feeling
- turn common problem types into clear mental models
- practice patterns instead of memorizing random solutions

## How to use it

- start with `01-learning-roadmap.md`
- read `02-problem-solving-framework.md`
- keep `03-patterns-cheat-sheet.md` nearby while practicing
- use `04-big-o-for-humans.md` when complexity starts feeling abstract
- read `05-coding-challenges-mindset.md` before practice sessions
