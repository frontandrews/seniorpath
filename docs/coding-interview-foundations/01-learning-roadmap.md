# Learning Roadmap

## Stage 1: Stop panicking

Focus on:

- arrays
- strings
- hashmaps
- sets

Goal:

- recognize the problem shape quickly
- stop brute forcing every problem with nested loops

## Stage 2: Learn the first real patterns

Focus on:

- two pointers
- sliding window
- stack
- queue

Goal:

- understand why the pattern saves work
- know the clues that hint at each pattern

## Stage 3: Learn structured search

Focus on:

- binary search
- recursion basics
- tree / graph traversal later

Goal:

- stop seeing binary search as magic
- understand search-space reduction

## Stage 4: Build repetition

For each topic:

- do a few easy problems
- explain the pattern in plain English
- only then move to medium problems

## Rule

Do not try to become good by reading ten solutions in a row.
Solve slowly, explain why the pattern works, then repeat.
